Linux/Ubuntu 下为 Android 创建无线分享热点

2015-06-24

文章目录

create ap hotspot for ubuntu linux

准备

https://w1.fi/hostapd/

https://github.com/oblique/create_ap

编译 hostapd

sudo apt-get install libnl-3-dev libnl-gen-3-dev
make && sudo make install

安装 create_ap

git clone https://github.com/oblique/create_ap.git
cd create_ap && make install

创建无线热点

sudo create_ap wlan0 ppp0 hotspot password
